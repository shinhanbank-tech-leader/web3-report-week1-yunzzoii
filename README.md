# 제목: 블록체인의 신뢰 형성 구조와 기존 금융 시스템의 신뢰 구조 비교

## 부서명: 정보서비스개발부
## 이름: 윤정

-------------------

## 레포트 내용
전통적인 금융 시스템과 블록체인 기반의 탈중앙화 시스템은 모두 신뢰를 기반으로 하지만, 그 신뢰를 구축하고 유지하는 방식에서는 근본적인 차이를 보입니다.

1. 기존 금융 시스템: 기관 중심의 신뢰
기존 금융 시스템은 국가가 공인한 제3의 신뢰 기관을 기반으로 운영됩니다.
중앙 집중식 구조: 은행, 카드사, 중앙은행 등 중앙화된 주체가 모든 거래 내역(원장)을 독점적으로 관리합니다.
신뢰의 근거: 법적 규제, 정부의 보증, 그리고 해당 기관의 브랜드 평판에 의존합니다. 
검증 방식: 거래의 유효성은 중앙 기관의 내부 전산 시스템에서 검증되며, 사용자는 그 과정을 투명하게 확인할 수 없습니다.
취약점: 중앙 서버가 해킹당하거나, 기관 내부의 부정부패가 발생할 경우 신뢰가 순식간에 붕괴될 위험이 있습니다.

2. 블록체인 시스템: 기술 중심의 신뢰
블록체인은 특정 기관 대신 암호학적 알고리즘과 합의 알고리즘을 통해 신뢰를 형성합니다.
분산형 구조 (P2P): 중앙 서버 없이 네트워크에 참여하는 모든 노드가 동일한 거래 장부를 공유하고 보관합니다.
신뢰의 근거: 수학적 증명과 코드의 불변성에 기반합니다.
검증 방식 (합의 알고리즘): 작업 증명(PoW)이나 지분 증명(PoS)과 같은 합의 알고리즘을 통해 다수의 참여자가 거래의 정당성을 검증합니다. 
한 번 기록된 데이터는 수정이 거의 불가능합니다.
투명성: 누구나 블록체인 익스플로러를 통해 실시간으로 거래 내역을 추적할 수 있어, 정보의 비대칭성이 해소됩니다.

기존 시스템이 사회적 합의와 법적 강제력을 통해 신뢰를 확보했다면, 블록체인은 기술적 무결성과 투명성을 통해 신뢰를 확보하고 있습니다.
